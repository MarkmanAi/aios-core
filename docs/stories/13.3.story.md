# Story 13.3 ‚Äî BobStatusWriter: Dashboard Bridge

## Story Statement

**As a** AIOS operator in Bob mode,
**I want** Bob's orchestration state written to a dashboard JSON file and emitted as events,
**so that** the dashboard displays real-time status without polling Bob's internal state directly.

## Acceptance Criteria

### AC-1: Status Write
- [x] `BobStatusWriter.write(status)` writes valid JSON to `.aios/dashboard/bob-status.json`
- [x] Directory is created if it does not exist; if creation fails, `write()` throws a descriptive `Error` (does not silently fail)
- [x] Write is atomic (temp file + rename pattern, Windows-safe fallback)
- [x] Concurrent calls to `write()` are serialized ‚Äî last write wins, no interleaved/corrupted output

### AC-2: Schema Validation
- [x] Written JSON conforms to Bob Status Schema: `version`, `phase`, `agent`, `progress` (0‚Äì100), `timestamp` (ISO 8601)
- [x] Invalid schema inputs cause `write()` to throw a `TypeError` with message describing the failing field ‚Äî no partial write occurs

### AC-3: Dashboard Event Emission
- [x] Each successful write triggers an event on the `DashboardEmitter`
- [x] Event type matches `BobEventTypes.STATUS_UPDATE`
- [x] Event payload contains the full written status object
- [x] If event emission fails (emitter throws), the error is logged but does NOT propagate ‚Äî file write result is returned to caller regardless

### AC-4: Observability Panel Integration
- [x] `ObservabilityPanel.setPipelineStage(stage)` called on each Bob phase change
- [x] Panel mode (`MINIMAL` / `DETAILED`) set based on educational mode flag
- [x] Panel refresh rate is hardcoded to 1000ms ‚Äî not user-configurable in this story scope

### AC-5: Test Coverage
- [x] `tests/core/orchestration/bob-status-writer.test.js` covers write, schema validation, event emission, dir creation failure, and emitter failure isolation
- [x] Dashboard emitter mock confirms event type and payload
- [x] Test verifies emitter failure does not propagate to caller
- [x] Coverage >= 85% on `bob-status-writer.js`

### Bob Status Schema
```javascript
{
  version: string,    // e.g. "1.0.0"
  phase: string,      // current pipeline phase
  agent: string,      // active agent name
  progress: number,   // 0-100
  timestamp: string,  // ISO 8601
}
```

### Atomic Write Pattern (Windows-safe)
```javascript
const tmpPath = `${targetPath}.tmp.${process.pid}`;
await fs.writeFile(tmpPath, JSON.stringify(status));
try { await fs.rename(tmpPath, targetPath); }
catch { await fs.writeFile(targetPath, JSON.stringify(status)); }
```
