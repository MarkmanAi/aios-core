# Story 13.7 ‚Äî `aios bob`: Bob CLI Entry Point

## Story Statement

**As a** Bob mode user,
**I want** to run `aios bob` in my terminal and see what Bob can do,
**so that** I have a clear, discoverable entry point to interact with the system.

## Acceptance Criteria

### AC-1: bin/aios.js Registration
- [ ] `case 'bob':` is added to `bin/aios.js` command switch
- [ ] Loads `src/bob/cli.js` and calls `createBobCommand().parseAsync(...)`
- [ ] Pattern matches existing `case 'video':` (Story 7.1 AD-2)

### AC-2: Help Display (no subcommand)
- [ ] `aios bob` with no subcommand prints Bob's help
- [ ] Help lists all 6 subcommands: `status`, `do`, `stop`, `resume`, `explain`, `switch`
- [ ] Each subcommand has a one-line description

### AC-3: `--help` Flag
- [ ] `aios bob --help` shows the same help as `aios bob` with no args
- [ ] Output includes usage example: `aios bob do "implement login feature"`

### AC-4: AIOS Installation Check
- [ ] If `.aios-core/` is not found in `process.cwd()`, prints error and exits with code 1
- [ ] Error message: `"Bob requires AIOS to be installed. Run: npx @synkra/aios-install"`

### AC-5: Profile Guard
- [ ] Command works when `user_profile === 'bob'` (default)
- [ ] Command also works when `--force-bob` flag is passed (for testing/advanced users)
- [ ] Does NOT block advanced mode users (no profile check ‚Äî anyone can run `aios bob`)

### AC-6: Tests
- [ ] `src/bob/__tests__/cli.test.js` covers: no-args help, --help flag, missing .aios-core error
- [ ] `bin/aios.js` routing test: `bob` case loads correct module
- [ ] Coverage >= 80% on `src/bob/cli.js`

### cli.js Pattern
```javascript
const { Command } = require('commander');

function createBobCommand() {
  const bob = new Command('bob')
    .description('Bob ‚Äî your simplified AIOS interface')
    .addHelpText('after', '\nExample:\n  aios bob do "implement login feature"');

  // Register subcommands (Stories 13.8‚Äì13.12)
  bob.addCommand(require('./commands/status'));
  bob.addCommand(require('./commands/do'));
  // ... etc

  // Default: show help if no subcommand
  bob.action(() => bob.help());

  return bob;
}

module.exports = { createBobCommand };
```

### Installation Guard
```javascript
const fs = require('fs');
const path = require('path');

function requireAiosInstalled() {
  const aiosCoreDir = path.join(process.cwd(), '.aios-core');
  if (!fs.existsSync(aiosCoreDir)) {
    console.error('Bob requires AIOS to be installed. Run: npx @synkra/aios-install');
    process.exit(1);
  }
}
```
