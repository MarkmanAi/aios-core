### yaml → js-yaml Migration (self-learner.js ONLY)

`self-learner.js` uses `require('yaml')` (line ~25). Apply same pattern as Story 10.2:
```javascript
require('yaml')   → require('js-yaml')
yaml.parse(str)   → yaml.load(str)
yaml.stringify(o) → yaml.dump(o)
```

**Note:** `synapse-memory-provider.js` does **not** use yaml — no migration needed for that file.

### Session Cache Design (@architect decision 2026-03-02)

Cache key: `${agentId}-${bracket}` (e.g., `dev-MODERATE`).
- In-memory Map — resets automatically at session boundary (process restart)
- No explicit invalidation needed between agent switches
- `clearCache()` method exists for manual reset via `*synapse-diagnose`
[Source: @architect gate decision, PRD-SYNAPSE-001 v0.2 Section 4]
