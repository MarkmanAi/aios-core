# Story 24.2 — Memory Injection on Agent Activation

## Story

**As an** agent activating in Claude Code,
**I want** memories written by `MemoryWriter` to be loaded and injected into my activation context,
**so that** I can leverage institutional memory (corrections, patterns, heuristics) accumulated from past sessions from the moment I start.

## Context

`SynapseMemoryProvider` and `MemoryLoader` (both from Epic 16) exist and have correct APIs — but the bridge to agent activation is missing.

**ADR Decision 2 (Aria, 2026-03-15):** `UnifiedActivationPipeline._runPipeline()` currently loads 5 parallel data sources in Phase 1 (AgentConfig, SessionContext, ProjectStatus, GitConfig, PermissionMode). `MemoryLoader` must be added as the 6th parallel loader via the existing `_safeLoad` mechanism. The assembled `enrichedContext` must include a `memories` field.

Current Phase 1 (5 loaders):
```javascript
const [agentComplete, sessionContext, projectStatus, gitConfig, permissionData] =
  await Promise.all([...5 loaders...])
```

Target Phase 1 after this story (6 loaders):
```javascript
const [agentComplete, sessionContext, projectStatus, gitConfig, permissionData, memoryResult] =
  await Promise.all([
    // ... existing 5 loaders (unchanged) ...
    this._safeLoad('MemoryLoader', () => {
      const loader = new MemoryLoader(this.projectRoot)
      return loader.loadForAgent(agentId, { budget: 2000, layers: [1, 2] })
    }),
  ])
```

And `enrichedContext` must include `memories`:
```javascript
const enrichedContext = {
  // ... existing fields (unchanged) ...
  memories: memoryResult?.memories || [],   // NEW
}
```

**CRITICAL IMPORT RULE:** Always import from `core/synapse/memory/` — never from `core/memory-intelligence/`.

### _safeLoad pattern (already in file)

```javascript
// Existing pattern in unified-activation-pipeline.js:
this._safeLoad = async (name, loader) => {
  try {
    return await Promise.race([
      loader(),
      new Promise((_, reject) =>
        setTimeout(() => reject(new Error(`${name} timeout`)), 150)
      ),
    ]);
  } catch (err) {
    // Silent degradation — return null on any failure
    process.stderr.write(`[UAP] ${name} failed: ${err.message}\n`);
    return null;
  }
};
```
