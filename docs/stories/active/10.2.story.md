### yaml → js-yaml Migration (REQUIRED)

The source file uses `const yaml = require('yaml')` (line ~13). The repo uses `js-yaml`.

**Step 1 — Change require:**
```javascript
// FROM:
const yaml = require('yaml');
// TO:
const yaml = require('js-yaml');
```

**Step 2 — Update API calls:**
```javascript
// yaml.parse() → yaml.load()
const data = yaml.parse(fileContent);  // FROM
const data = yaml.load(fileContent);   // TO

// yaml.stringify() → yaml.dump()
const str = yaml.stringify(data);  // FROM (verify if used)
const str = yaml.dump(data);       // TO
```

Run `grep -n "yaml\." .aios-core/core/synapse/memory/memory-index.js` after copy to find all usages.
