# Story 24.1 ‚Äî SelfLearner Trigger

## Story

**As a** SYNAPSE pipeline,
**I want** `SelfLearner.run()` to be called automatically after `extractSessionDigest()` completes on every context compaction,
**so that** session digests are processed into institutional memory without requiring any manual intervention.

## Context

`synapse-precompact.js` currently calls `extractSessionDigest(context)` inside a `setImmediate` fire-and-forget block ‚Äî and stops there. `SelfLearner.run()` exists (Epic 16, 395 tests passing) but is never invoked in the automated pipeline.

**ADR Decision 1 (Aria, 2026-03-15):** Extend the existing `synapse-precompact.js` hook by chaining `SelfLearner.run(projectDir)` immediately after `extractSessionDigest()` in the same `setImmediate` async chain. This is the natural extension ‚Äî no new infrastructure, no new hook types.

Execution sequence after this story:
```javascript
setImmediate(async () => {
  try {
    await extractSessionDigest(context)       // writes digest to .aios/session-digests/
    await SelfLearner.run(context.projectDir) // reads digest ‚Üí MemoryWriter.write()
  } catch (err) {
    process.stderr.write(`[SYNAPSE] Pipeline error: ${err.message}\n`)
    // compact is NEVER blocked
  }
})
// hook returns here immediately ‚Äî well within 50ms budget
```

**CRITICAL IMPORT RULE:** Always import from `.aios-core/core/synapse/memory/` ‚Äî never from `.aios-core/core/memory-intelligence/`.

**Blocked by:** Story 24.0 merged to remote.

Consistent with the existing `extractSessionDigest` require pattern ‚Äî modules are loaded lazily to avoid startup cost. Hook returns to Claude Code in < 50ms; modules load asynchronously.
