# Story 24.1 — SelfLearner Trigger

## Story

**As a** SYNAPSE pipeline,
**I want** `SelfLearner.run()` to be called automatically after `extractSessionDigest()` completes on every context compaction,
**so that** session digests are processed into institutional memory without requiring any manual intervention.

## Context

`synapse-precompact.js` currently calls `extractSessionDigest(context)` inside a `setImmediate` fire-and-forget block — and stops there. `SelfLearner.run()` exists (Epic 16, 395 tests passing) but is never invoked in the automated pipeline.

**ADR Decision 1 (Aria, 2026-03-15):** Extend the existing `synapse-precompact.js` hook by chaining `SelfLearner.run(projectDir)` immediately after `extractSessionDigest()` in the same `setImmediate` async chain. This is the natural extension — no new infrastructure, no new hook types.

Execution sequence after this story:
```javascript
setImmediate(async () => {
  try {
    await extractSessionDigest(context)       // writes digest to .aios/session-digests/
    await SelfLearner.run(context.projectDir) // reads digest → MemoryWriter.write()
  } catch (err) {
    process.stderr.write(`[SYNAPSE] Pipeline error: ${err.message}\n`)
    // compact is NEVER blocked
  }
})
// hook returns here immediately — well within 50ms budget
```

**CRITICAL IMPORT RULE:** Always import from `.aios-core/core/synapse/memory/` — never from `.aios-core/core/memory-intelligence/`.

### Critical Issue — AC-2 Broken at Runtime

**Root cause:** The story Dev Notes described an incorrect API. `self-learner.js` exports a named export object, not the class directly. `run()` is an instance method, not static.

**What happens at runtime (every compaction):**
```
[SYNAPSE] Pipeline error: SelfLearner.run is not a function
```
SelfLearner never executes. The story's core feature is a silent no-op.

**Fix required in `.claude/hooks/synapse-precompact.js`:**
```javascript
// WRONG (current):
const SelfLearner = require(SELF_LEARNER_PATH);
await SelfLearner.run(context.projectDir);

// CORRECT:
const { SelfLearner } = require(SELF_LEARNER_PATH);
const learner = new SelfLearner(context.projectDir);
await learner.run();
```

**Fix required in `tests/unit/hooks/synapse-precompact.test.js`:**
```javascript
// WRONG (current) — mock matches wrong API:
jest.doMock(SELF_LEARNER_PATH, () => ({
  run: mockSelfLearnerRun,
}));

// CORRECT — mock must reflect named export + constructor pattern:
const MockSelfLearner = jest.fn().mockImplementation(() => ({
  run: mockSelfLearnerRun,
}));
jest.doMock(SELF_LEARNER_PATH, () => ({ SelfLearner: MockSelfLearner }));
// Add assertion: expect(MockSelfLearner).toHaveBeenCalledWith(process.cwd())
```

**Reference:** `tests/unit/synapse/self-learner-writer-integration.test.js:15` shows the correct import: `const { SelfLearner } = require(...)`.
