## Dev Notes

### Context

This story fixes 4 mechanical issues discovered when CI first ran on PR #6 (Story 10.6):

**Issue 1 — Orphaned License Tests (9 suites)**
- Path: `tests/license/*.test.js`
- Root cause: `pro/license/` module does not exist in this repo. Tests were written for a "pro" tier feature that lives in a separate private repository or has not been implemented yet.
- User confirmed (2026-03-03): `pro/license` will not be implemented in this repo → tests are dead debt
- Fix: Add to `testPathIgnorePatterns` in `jest.config.js`

**Issue 2 — Orphaned MMOS Squad Tests (5 suites)**
- Path: `squads/mmos-squad/tests/**/*.test.js`
- Root cause: Tests reference `.aios-core/infrastructure/services/{clickup,google-drive,file-service}` which do not exist. The squad was created but the backing infrastructure services were never implemented.
- Fix: Add to `testPathIgnorePatterns` in `jest.config.js`

**Issue 3 — WorkflowExecutor Not Exported**
- File: `.aios-core/core/orchestration/index.js`
- Root cause: `workflow-executor.js` exists at `.aios-core/core/orchestration/workflow-executor.js` but was never added to the module's `module.exports`. One test (`tests/core/workflow-executor.test.js`) fails because it verifies the export exists.
- Fix: Add `const WorkflowExecutor = require('./workflow-executor')` and include in `module.exports`

**Issue 4 — IDE Sync Drift (5 files)**
- Root cause: Agent definition files in `.aios-core/development/agents/` were modified but the synced copies in `.claude/commands/AIOS/agents/` were not updated.
- Fix: `npm run sync:ide` then commit the generated files

**Issue 5 — Install Manifest Outdated**
- Root cause: 109 new files added to `.aios-core/` and 45 files modified, but `scripts/install-manifest.json` (or equivalent) was not regenerated.
- Fix: `npm run generate:manifest` then commit
