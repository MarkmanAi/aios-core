# Story 13.9 — `aios bob do "<request>"`: Natural Language Task Delegation

## Story Statement

**As a** Bob mode user,
**I want** to type `aios bob do "implement the login feature"` and have Bob figure out what to do,
**so that** I never need to know which agent to call or how to structure a development request.

## Acceptance Criteria

### AC-1: Command Invocation
- [x] `aios bob do "<natural language request>"` is a valid command
- [x] `--request` flag is an alias: `aios bob do --request "implement login"`
- [x] Missing request argument shows usage error: `"Usage: aios bob do \"<your request>\""`

### AC-2: Bob's Routing via BobOrchestrator
- [x] User input is passed as `{ userGoal: userInput }` to `BobOrchestrator.orchestrate()`

### orchestrate() Return Value — Action Types

[Source: `.aios-core/core/orchestration/bob-orchestrator.js` lines 413-535]

`orchestrate()` returns an `OrchestrationResult`:
```javascript
// Shape:
{
  success: boolean,
  projectState: string|null,     // ProjectState enum value
  action: string,                // What happened — see action types below
  data: Object,                  // Action-specific payload
  error: string,                 // Only on failure
  cleanupResult: Object,         // From DataLifecycleManager
}
```

**Action types and required CLI response:**

| `action` | Meaning | CLI Must Do |
|----------|---------|-------------|
| `educational_mode_toggle` | User input was a toggle command | Show persistence prompt, call `handleEducationalModeToggle(enable, type)` |
| `resume_prompt` | Existing session found | Show 4-option prompt, call `handleSessionResume(option)` |
| `ask_objective` | Project ready, what to build? | Show options menu, loop back with selected story |
| `story_executed` | Story ran to completion | Print success summary |
| `onboarding` | No AIOS config | Print install instruction |
| `lock_failed` | Another Bob running | Print error, exit 1 |
| `error` | Orchestration error | Print error, exit 1 |

### Multi-Step Action Loop Implementation

The CLI must handle the case where `orchestrate()` returns a prompt action. Pattern:

```javascript
async function runBobDo(userInput, options) {
  const spinner = ora('Bob is thinking...').start();
  try {
    const orchestrator = new BobOrchestrator(process.cwd());
    const result = await orchestrator.orchestrate({ userGoal: userInput });
    spinner.stop();

    // Handle action types
    switch (result.action) {
      case 'lock_failed':
        console.error(chalk.red('Bob is already running. Check .aios/locks/ or wait.'));
        process.exit(1);
        break;

      case 'onboarding':
        console.log('Bob requires AIOS setup. Run: npx @synkra/aios-install');
        break;

      case 'educational_mode_toggle': {
        // Show persistence prompt and apply toggle
        const { enable } = result.data;
        const answer = await inquirer.prompt([{
          type: 'list',
          name: 'persistence',
          message: result.data.persistencePrompt,
          choices: ['session', 'permanent'],
        }]);
        await orchestrator.handleEducationalModeToggle(enable, answer.persistence);
        console.log(chalk.green(`Educational mode ${enable ? 'ON' : 'OFF'} (${answer.persistence})`));
        break;
      }

      case 'resume_prompt': {
        const answer = await inquirer.prompt([{
          type: 'list',
          name: 'option',
          message: result.data.formattedMessage,
          choices: ['continue', 'review', 'restart', 'discard'],
        }]);
        const resumeResult = await orchestrator.handleSessionResume(answer.option);
        printCompletionSummary(resumeResult);
        break;
      }

      case 'ask_objective': {
        const answer = await inquirer.prompt([{
          type: 'list',
          name: 'objective',
          message: 'What do you want to do?',
          choices: result.data.options,
        }]);
        // Re-run with storyPath if user selects a story directly
        // (simplified: just print the selection — full routing out of scope for this story)
        console.log(chalk.blue(`Bob will handle: ${answer.objective}`));
        break;
      }

      case 'story_executed':
        printCompletionSummary(result);
        break;

      default:
        if (!result.success) {
          console.error(chalk.red(`Bob: ${result.error || 'Unknown error'}`));
          process.exit(1);
        }
        printCompletionSummary(result);
    }
  } catch (err) {
    spinner.fail(`Bob encountered an error: ${err.message}`);
    process.exit(1);
  }
}
```

### Completion Summary Format

```javascript
function printCompletionSummary(result) {
  if (!result.success) {
    console.error(chalk.red(`Bob: ${result.error || 'Something went wrong'}`));
    return;
  }
  console.log(chalk.green('\nBob here. Done.'));
  console.log(`What happened: ${result.action}`);
  console.log(`Next: Run "aios bob status" to see where things stand`);
}
```
