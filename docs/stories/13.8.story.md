# Story 13.8 — `aios bob status`: Project Status

## Story Statement

**As a** Bob mode user,
**I want** to run `aios bob status` and see where my project stands in plain language,
**so that** I know what Bob is doing and what the next step is — without reading JSON files.

## Acceptance Criteria

### AC-1: Basic Status Output
- [x] Command outputs: project name, detected project state, active story (if any), active agent (if any), last activity time
- [x] Output is formatted in Bob's voice (plain language, no technical jargon)

### AC-2: Active Story Display
- [x] Reads active story from `.aios/project-status.yaml`
- [x] Shows story ID, title, and current task if in progress
- [x] Shows "No active story" if none in progress

### AC-3: Active Agent Display
- [x] Shows which agent is currently running (if any) from `bob-status.json`
- [x] Shows agent's friendly name (e.g., "Dex" not "@dev")
- [x] Shows "No agent active" if idle

### AC-4: Educational Mode Adaptation
- [x] When educational mode is OFF: concise 5-line output
- [x] When educational mode is ON: adds explanation of each state + what Bob will do next

### AC-5: Nothing Active State
- [x] When no story is active AND no agent is running: shows encouraging message
- [x] Message includes next action suggestion: `"Run: aios bob do \"<your request>\""`

### AC-6: Tests
- [x] `src/bob/__tests__/status.test.js` covers: active story, no story, active agent, no agent, educational mode variants
- [x] Coverage >= 80% on `src/bob/commands/status.js`

function timeAgo(isoTimestamp) {
  const diffMs = Date.now() - new Date(isoTimestamp).getTime();
  const diffMin = Math.floor(diffMs / 60000);
  if (diffMin < 1) return 'just now';
  if (diffMin === 1) return '1 minute ago';
